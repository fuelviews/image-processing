    ```bash

    ```

-------------------

## Getting Started

### Local usage

1. Place the images in the `images` folder
2. Run

    ```bash
    npm run images
    ```

3. Grab the images from `public/images`

### In browser

1. Run

    ```bash
    netlify dev
    ```

2. Open [https://localhost:8888](https://localhost:8888)
